Detailed Description

A client that uses a shared connection by wrapping a real client.

A shared client is useful when the proxy is to be used in different contexts within the same application process, where it's undesirable to create multiple clients and therefore multiple connections to the same server. This class wraps a real client and orchestrates method calls in such a way that multiple shared clients can act independently and concurrently using the same underlying connection.
