enum loon::LogLevel
strong

Severity levels for log messages.

Severity increases with the integer value of the log level.

Enumerator
Debug 
Info 

Messages to aid with debugging.

Warning 

Informational messages, like served requests.

Error 

Events and errors that can be handled gracefully.

Fatal 

Errors which cause the client to restart the connection.

Silent 

Errors which cause the client to fail and stop.
